# 機能一覧

| 項目 | 内容 |
|------|------|
| 文書バージョン | 1.0 |
| 作成日 | 2026-06-29 |
| 作成者 | Higasizono |

---

## 1. ボード管理

| # | 機能 | 詳細 |
|---|------|------|
| B-01 | ボード一覧表示 | トップページにボードのカード一覧を表示する |
| B-02 | ボード作成 | ボード名を入力して新規ボードを作成できる |
| B-03 | ボード削除 | ボードを削除できる（配下のカラム・カードも全て削除される） |
| B-04 | ボード選択 | ボードをクリックするとボード詳細ページに遷移する |

## 2. カラム管理

| # | 機能 | 詳細 |
|---|------|------|
| C-01 | カラム表示 | ボード内のカラムを横並びで表示する |
| C-02 | カラム追加 | カラム名を入力して新しいカラムをボードに追加できる |
| C-03 | カラム削除 | カラムを削除できる（配下のカードも全て削除される） |

## 3. カード管理

| # | 機能 | 詳細 |
|---|------|------|
| K-01 | カード表示 | カラム内のカードを縦に一覧表示する |
| K-02 | カード追加 | タイトルを入力してカラムにカードを追加できる |
| K-03 | カード編集 | カードのタイトルをクリックして編集できる |
| K-04 | カード削除 | カードを削除できる |
| K-05 | カード移動 | ドラッグ&ドロップでカードを別カラムに移動できる |

## 4. データ保存

| # | 機能 | 詳細 |
|---|------|------|
| D-01 | 自動保存 | 操作のたびにlocalStorageへ自動保存する |
| D-02 | データ復元 | ページ読み込み時にlocalStorageからデータを復元する |
| D-03 | 初期データ | 初回起動時はデフォルトのボードとカラムを表示する |
