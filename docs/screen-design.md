# 画面設計書

---

## 1. トップページ (`/`)

### 1-1. 概要

| 項目 | 内容 |
|------|------|
| URL | `/` |
| 役割 | ボードの一覧表示・作成・削除 |
| 初期状態 | APIから取得したボードが0件の場合は空状態を表示 |

---

### 1-2. コンポーネント構成

```
<App>
└── <TopPage>
    ├── <Header>
    │   └── アプリ名テキスト
    └── <BoardList>
        ├── <BoardCard> × N（ボード数分）
        │   ├── ボード名テキスト（クリックでボードページへ遷移）
        │   └── <DeleteButton>（「×」ボタン）
        ├── <EmptyState>（ボードが0件のとき表示）
        └── <CreateBoardButton>
            └── <CreateBoardForm>（作成ボタンクリック時に切り替わる）
                ├── <input>（テキスト入力）
                ├── <SubmitButton>（「作成」）
                └── <ErrorMessage>（バリデーションエラー時）
```

### 1-4. インタラクション仕様

| 操作 | 挙動 |
|------|------|
| ボードカードをクリック | `/board/:id` に遷移 |
| 「×」ボタンをクリック | 削除確認ダイアログを表示 |
| 「＋ 新規ボードを作成」クリック | インライン入力フォームに切り替わる。フォーカスが自動的に入力フィールドへ移動 |
| Enter（フォーム内） | バリデーション → 成功時：ボード作成・一覧更新。失敗時：エラー表示 |
| Esc（フォーム内） | フォームを閉じる |

## 2. ボードページ (`/board/:id`)

### 2-1. 概要

| 項目 | 内容 |
|------|------|
| URL | `/board/:id` |
| 役割 | カラムとカードの表示・カードの追加・編集・削除・並び替え |
| 初期状態 | 「未着手」「進行中」「完了」の3カラムを常に表示（0件になることはない） |

---

### 2-2. コンポーネント構成

```
<App>
└── <BoardPage>
    ├── <Header>
    │   ├── <BackButton>（「← 戻る」）
    │   └── ボード名テキスト
    └── <ColumnList>（横スクロールコンテナ）
        └── <Column> × 3（未着手・進行中・完了に固定）
            ├── <ColumnHeader>
            │   └── カラム名テキスト
            ├── <CardList>（dnd-kit の SortableContext）
            │   ├── <Card> × N（カード数分）
            │   │   ├── カード名テキスト（クリックで編集モード）
            │   │   ├── <CardEditForm>（編集モード時に表示）
            │   │   │   ├── <input>
            │   │   │   └── <ErrorMessage>
            │   │   └── <DeleteButton>（「×」ボタン）
            │   └── <EmptyCardState>（カードが0件のとき）
            └── <AddCardButton>
                └── <AddCardForm>（クリック時に展開）
                    ├── <textarea> または <input>
                    ├── <SubmitButton>（「追加」）
                    ├── <CancelButton>（「キャンセル」）
                    └── <ErrorMessage>
```

### 2-4. インタラクション仕様

| 操作 | 挙動 |
|------|------|
| 「← 戻る」クリック | トップページへ遷移 |
| カードタイトルをクリック | 編集モードに切り替わる。テキスト全選択状態 |
| カード「×」クリック | 確認なしで即時削除 |
| 「＋ カードを追加」クリック | カラム内にフォームを展開。自動フォーカス |
| カードをドラッグ | ゴースト表示。移動先カラムとドロップ位置をハイライト |
| Enter（各フォーム内） | バリデーション後に保存 |
| Esc（各フォーム・編集内） | キャンセル・元に戻す |
| ページをスクロール（横） | カラムが画面幅を超えた場合、横スクロールで表示 |
