## 1. 画面一覧

| 画面名 | URL | 説明 |
|--------|-----|------|
| トップページ | `/` | ボード一覧を表示・作成・削除 |
| ボードページ | `/board/:id` | カラムとカードを表示・操作 |

---

## 2. トップページ (`/`)

### 2-1. 表示要件

| 項目 | 内容 |
|------|------|
| URL | `/` |
| 役割 | ボードの一覧表示・作成・削除 |
| 初期状態 | localStorageにデータがない場合は空状態を表示 |

- ボードが1件以上ある場合、ボードカードをグリッドで一覧表示する
- ボードが0件の場合、空状態メッセージ（「まだボードがありません」）と作成ボタンを表示する
- 常に「＋ 新規ボードを作成」ボタンを表示する

### 2-2. インタラクション仕様

| 操作 | 挙動 |
|------|------|
| ボードカードをクリック | `/board/:id` に遷移 |
| 「×」ボタンをクリック | 削除確認ダイアログを表示 |
| 「＋ 新規ボードを作成」クリック | インライン入力フォームに切り替わる。フォーカスが自動的に入力フィールドへ移動 |
| Enter（フォーム内） | バリデーション → 成功時：ボード作成・一覧更新。失敗時：エラー表示 |
| Esc（フォーム内） | フォームを閉じる |

---

## 3. ボードページ (`/board/:id`)

### 3-1. 表示要件

| 項目 | 内容 |
|------|------|
| URL | `/board/:id` |
| 役割 | カラムとカードの表示・カードの追加・編集・削除・並び替え |
| 初期状態 | 「未着手」「進行中」「完了」の3カラムを常に表示（0件になることはない） |

- ヘッダーに「← 戻る」リンクとボード名を表示する
- カラムを横並びで表示する。カラムが画面幅を超えた場合は横スクロールで対応する
- カラムは「未着手」「進行中」「完了」の3種に固定。追加・削除・名称変更のUIは提供しない

### 3-2. インタラクション仕様

| 操作 | 挙動 |
|------|------|
| 「← 戻る」クリック | トップページへ遷移 |
| カードタイトルをクリック | 編集モードに切り替わる。テキスト全選択状態 |
| カード「×」クリック | 確認なしで即時削除 |
| 「＋ カードを追加」クリック | カラム内にフォームを展開。自動フォーカス |
| カードをドラッグ | ゴースト表示。移動先カラムとドロップ位置をハイライト |
| Enter（各フォーム内） | バリデーション後に保存 |
| Esc（各フォーム・編集内） | キャンセル・元に戻す |
| ページをスクロール（横） | カラムが画面幅を超えた場合、横スクロールで表示 |
